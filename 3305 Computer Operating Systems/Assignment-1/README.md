# Parallel and Serial Prime Computations  

	We were tasked to make a program that calculates the sum of primes of each quartile between two positive integers.   
	The program must be able to be able to complete this in a serial mode and parallel mode.  

## Installation  

	Download the .tar zip and compile and run the code in a linux enviroment  

## Usage  

	The use case is as follows: ./executeable mode min max  
	Where the mode is either 0 (serial) or 1(parallel)  
	And min and max are valid positive integers where min < max
