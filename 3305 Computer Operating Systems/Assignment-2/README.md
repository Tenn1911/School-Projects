# Mathematical Formula Computations VIA Child Processes  

	This program will compute a mathematical formula on 2 four digit numbers using mutliple child processes to do the heavy lifting  

## Installation   

	Download the .tar zip and compile and run the code in a linux enviroment  

## Usage  

    When compiling this code use -lm at the end of the command to link the math library  

    The use case is as follows: ./executeable  positive_integer positive_integer  
    Each positive integer must be a four digit number
