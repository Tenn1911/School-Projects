# 9x9 Sudoku Validator  

	This program will validate a 9x9 Sudoku board VIA threads   

## Installation   

	Download the .tar zip and compile and run the code in a linux enviroment  

## Usage  

    To compile run the make command and the Makefile will generate an executable called asn3   
    The use case is as follows: ./asn3 infile  
    Where infile is a text file containg a 9x9 Sudoku grid, where the numbers are seperated by spaces  
    An example input is in the .tar zip
